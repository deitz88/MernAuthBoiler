## Setup 

1. Clone the repo
2. setup Git repo


#### Setup your git repo

```
git init
git add .
git commit -m "first commit"
git remote add origin git@github.com:<nusername>/<gitfilename>.git // this will be whatever your address will be
git push -u origin master
```

#### Setup the App

```npm install```
